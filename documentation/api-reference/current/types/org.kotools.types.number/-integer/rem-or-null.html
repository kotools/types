remOrNull

fun remOrNull(other: Integer): Integer?

Returns the Euclidean remainder of dividing this integer by the other one, or returns null if the other integer is zero.

This function uses Euclidean division: the remainder is always non-negative and less than the absolute value of other (0 <= remainder < |other|), regardless of the sign of this integer.


Calling from Kotlin

Here's an example of calling this function from Kotlin code:

val x: Integer = Integer.fromLong(-7)
val y: Integer = Integer.fromLong(2)

val quotient: Integer? = x.divOrNull(y)
val remainder: Integer? = x.remOrNull(y)

checkNotNull(quotient)
checkNotNull(remainder)
check(quotient == Integer.fromLong(-4))
check(remainder == Integer.fromLong(1))
check(x == quotient * y + remainder)
