toInt

fun toInt(): Int

Returns the Int representation of this integer, or throws an ArithmeticException if this integer is out of the Int range, i.e., less than Int.MIN_VALUE or greater than Int.MAX_VALUE.


Calling from Kotlin

Here's an example of calling this function from Kotlin code:

val x: Integer = Integer.fromLong(42)
val result: Int = x.toInt()
check(result == 42)

val y: Integer = Integer.parse("9999999999")
val exception: Throwable? = runCatching { y.toInt() }
    .exceptionOrNull()
check(exception is ArithmeticException)

final Integer x = Integer.fromLong(42);

final Integer y = Integer.parse("9999999999");

    y.toInt();
